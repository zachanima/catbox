# Cat (in a) Box

Javascript game engine.



## Features

Implemented (work-in-progress):

* Rigidbodies
* Collision
  * Axis-aligned bounding box
  * Pixel
* Rendering
  * Sprites
  * Camera
* Timing
* Input
  * Keyboard
  * Mouse


Not yet implemented:

* Networking
* Physics forces
* Collision
  * Circle
  * Oriented bounding box
  * Convex hull
* Audio
  * Sound
  * Music
* Rendering
  * Text
  * Shapes
  * Particle system
  * GUI
